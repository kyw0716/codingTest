#57 숫자의 표현
<br> 연속수끼리 더해서 정해진 수를 갖는 경우의 수를 찾기! 연속되는 수니까 등차수열의 공비(?0가 1인 경우를 찾으면 되는것 같다. 이게 생각하면 복잡한데 등차수열의 일반식 a(n) = a + (n-1)d인데 여기서 d가 1이므로 수가 연속수가 추가될 때마다 1, 2, 3.. 이렇게 더해지는 것이다 그래서

```(java)
        while(n>0) {
            if(n%d==0) answer++; // n에 개수를 나눴을 때 나눠 떨어지면 주어진 수는 연속수들로 만들어질 수 있는 수
            n -= d; // a만 남겨서 나눠주기 위해 더해진 수를 n에서 빼어줌
            //ex. n = 15이고 a(1) = a, a(2) = a+1, a(3) = a+2
            //이때 n에서 1, 2만큼 빼고 계산하면 12 = 3a, a = 4이므로 나눠 떨어져 연속수들의 합으로 만들 수 있다!
            d++; //이게 n !!
        }
```
